What is Git & GitHub?

  • Git: A tool that tracks changes in your code and helps you manage different versions.
  • GitHub: A website that hosts Git repositories and allows you to share your code with others.
  • Why use them? Together, they help you collaborate, backup your work, and keep track of your code history!

Setting up Git

  1. Download Git from the official website: git-scm.com
  2. Install Git by following the installation wizard instructions (make sure to select the default options).
  3. Verify the installation:
    1. Open your terminal or command prompt.
    2. Type git --version and press Enter. You should see the installed version of Git.

Note: Remember to take screenshots at each step for clarity!

Creating a GitHub Account

  1. Go to the GitHub signup page: github.com/join
  2. Fill out the required information and follow the prompts to create your account.

Creating a Repository

  1. Once logged in to GitHub, click on the "+" icon in the upper right corner and select "New repository."
  2. Enter a repository name, such as js-journey-level1.
  3. Choose the visibility (public or private) and click "Create repository."

Cloning the Repository

  1. On your newly created repository page, click the green "Code" button.
  2. Copy the HTTPS URL provided.
  3. Open your terminal or command prompt and navigate to the folder where you want the repository.
  4. Type git clone [paste the URL] and press Enter.
